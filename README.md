# REACT 臨床推論レベル評価ツール

**Rapid Evaluation Assessment of Clinical Reasoning Tool**

理学療法士・作業療法士向けの臨床推論能力評価ツールです。

## 🎯 概要

5つのドメインで臨床推論能力を評価し、プログラム開始時（Before）と終了時（After）の成長を可視化します。

### 5つのドメイン

1. **Collecting（収集）** - データ収集・仮説志向型の情報収集
2. **Interpreting（解釈）** - 診断推論・鑑別診断の生成
3. **Managing（管理）** - マネジメント推論・評価と治療計画
4. **Communicating（コミュニケーション）** - 患者中心のケア・医療チームとの連携
5. **Reflecting（省察）** - メタ認知・臨床場面の振り返り

## 📊 評価方法

- 各ドメインを **1-3点** の3段階で評価
- **合計スコア**: 5点（最小）〜 15点（最大）
- レーダーチャートで視覚化

## 🚀 主な機能

- ✅ Before/After評価の比較
- ✅ レーダーチャートによる可視化
- ✅ CSVエクスポート
- ✅ Markdown形式の比較レポート出力
- ✅ ブラウザに自動保存（LocalStorage）
- ✅ オフラインで動作
- ✅ スマホ対応
